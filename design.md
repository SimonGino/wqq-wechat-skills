## 2) 如何提取爆文 DNA（可复用方法）

### 2.1 定义：什么是爆文 DNA

爆文 DNA 不是具体句子，而是可跨主题复用的结构：

1. 选题层：是否命中读者当下痛点
2. 标题层：是否承诺明确结果（读完能拿到什么）
3. 内容层：是否可执行（步骤、命令、验证、排错）
4. 传播层：是否有明确转发价值（能帮别人省时间）

### 2.2 拆解模板（每篇都用同一模板）

1. 开场：是否 3 句内说清痛点
2. 承诺：是否有“一句话结论/你会得到”
3. 目录：是否覆盖完整闭环（装、配、用、验、排错）
4. 步骤：是否每步有可执行动作
5. 验证：是否每步写清成功标准
6. 排错：是否覆盖高频失败场景
7. 附录：是否提供可复制资产（命令、配置、清单）
8. 转发触发：是否有明确转发对象（如“转给正在卡这一步的同事”）

### 2.3 提炼为一句可执行公式

`爆文公式 = 热点痛点 x 明确承诺 x 可执行闭环 x 转发价值`

### 2.4 发文前检查（不过不发）

- [ ] 标题包含结果导向词（从 0 到可用、完整指南、避坑）
- [ ] 前 150 字内给出“一句话结论”
- [ ] 有最小可跑通路径
- [ ] 每个关键步骤有验证方式
- [ ] 有常见坑 -> 对应修法
- [ ] 有可复制附录
- [ ] 结尾有明确转发对象

### 2.5 发文后复盘（24h 轻量版）

只回答 3 个问题：

1. 哪个模块最容易被转发（通常是清单/排错/附录）
2. 哪一步最容易卡住（决定下一篇选题）
3. 哪句承诺最被认可（决定下一篇标题语言）

复盘输出必须落地成两项：

- 下一篇文章题目
- 下一篇必须新增的一个模块
